# Vuetify docker-composeの初期起動
## 前提条件
- Docker がインストールされていること
- Docker Compose がインストールされていること

## プロジェクトセットアップ
```
git clone https://github.com/yoshis2/vue-project.git
```

### Dockerの設置しているフォルダに移動
```
cd docker
```

### Dockerを起動
```
docker-compose up
```

### 2、3分ほど待ってから以下URLへ遷移するとVue(Vuetify)画面が起動する
```
http://localhost:2300
```
